사용방법 가이드

--------------------------------------------------------------------------------------------------------------

1) 기본 입력값 설정

페이지 상단 입력칸에서 아래 항목을 입력합니다.

투자 시작 나이: 그래프 X축(나이) 계산에 사용됩니다.

초기 투자원금: 시작 시점에 한 번 투자하는 금액입니다.

매월 적립금: 적립식으로 매월 추가 투자하는 금액입니다.

적립식 투자 기간(년/월): 말 그대로 매월 적립식 투자를 계속하는 기간입니다.

거치기간(년/월): 적립식 투자를 끝낸 뒤, 추가 납입 없이 입력한 수익률로 자산을 그대로 굴리는 기간입니다.

연평균 수익률(%): 예상 수익률(연평균)을 입력합니다.

기간의 년/월은 합산으로 계산됩니다. (예: 2년 6개월 = 30개월)

✅ 참고

이 계산기는 추가매수를 “적립식 투자 기간 안에서만” 가능하도록 설정되어 있습니다.
(즉, 거치기간에 들어간 이후 월차의 추가매수는 기본 설정에서는 반영되지 않습니다.)

--------------------------------------------------------------------------------------------------------------

2) 추가매수 입력 (선택)

적립식 기간 중 특정 월에 목돈을 추가 투자하고 싶다면 추가매수에 입력합니다.

월차: 0 = 시작 시점, 1 = 1개월차, 24 = 24개월차 …

금액(원): 추가 투자할 금액을 입력합니다. (입력 중 자동으로 콤마가 붙습니다.)

필요하면 **“+ 행 추가”**로 여러 건을 입력할 수 있습니다.

--------------------------------------------------------------------------------------------------------------

3) 계산 실행

입력값을 모두 설정한 뒤:

“계산하기” 버튼을 누르면 결과가 갱신됩니다.

(현재 버전은) 입력값을 수정하면 자동으로 결과가 업데이트되며, 타이핑을 멈춘 뒤 잠깐 후(약 0.25초) 반영됩니다.

--------------------------------------------------------------------------------------------------------------

4) 결과 확인

결과 영역에서 아래 값을 확인할 수 있습니다.

총 투자원금(최종)

최종자산(추정)

최종이익

수익률(%)

또한, 추가매수 항목별로 “그 금액이 최종 시점에 얼마가 되는지(FV)”도 표로 확인할 수 있습니다.

--------------------------------------------------------------------------------------------------------------

5) 그래프/표 보기

그래프는 원금 + 이익(누적 막대) 형태로 표시됩니다.

X축은 나이 기준으로 표시됩니다.

아래 연차 범위 슬라이더로 원하는 구간만 골라서 확대해 볼 수 있습니다.

표에서는 연도별로 투자원금/이익/자산을 숫자와 함께 (약 1.2억) 같은 요약 표기로 확인할 수 있습니다.
